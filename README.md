# 基于Vue实现的数据可视化大屏

## 项目运行

### 安装
```
npm install
```

### 启动
```
npm run serve
```

### 编译
```
npm run build
```

### 检查
```
npm run lint
```

## 技术要点
- vue使用
- [flexible.js](https://github.com/amfe/lib-flexible) 与 rem 单位实现自适应效果
- echarts图标以及地图的使用
- css基础
- [datav](https://github.com/DataV-Team/Datav)应用
